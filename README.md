Simulador Digital de Lógica Computacional

Aplicação web desenvolvida com HTML, CSS e JavaScript para simular operações da lógica binária.

Permite inserir dois valores decimais (0–15), convertê-los para binário (4 bits) e aplicar as operações:

AND

OR

XOR

NOT

Os resultados são exibidos em binário e decimal, além de incluir tabela verdade para 1 bit.

Projeto acadêmico voltado à consolidação de conceitos de sistema binário e Álgebra Booleana, executado integralmente no navegador (client-side).
